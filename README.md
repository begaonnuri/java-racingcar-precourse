## 자동차 경주 게임

### 기능 요구사항

#### 입력
1. 자동차의 이름을 입력하는 기능
    1. 이름이 5자 초과인 경우 예외처리
1. 몇 번 이동할지 입력하는 기능

#### 중간과정 
1. 쉼표를 기준으로 이름을 구분하는 기능
1. 자동차에 이름을 부여하는(Car 객체를 생성하는) 기능
1. 0에서 9사이 random 값을 생성하는 기능
1. 전진할지 멈출지 결정하는 기능
1. 자동차를 움직이는 기능
    1. 전진하는 기능
    1. 멈추는 기능
        
#### 출력
1. 이동 결과를 출력하는 기능
1. 우승자들을 출력하는 기능
